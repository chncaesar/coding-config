# coding-config

Reusable Cursor rule files for coding, review, and pull request workflows.

## What's inside

- `rules/code-review.mdc`: guidance for thorough code reviews.
- `rules/code-style-python.mdc`: Python style, reuse, and testing rules.
- `rules/pull-request.mdc`: pull request body guidance.
- `rules/config-env-sync.mdc`: reminder to sync config changes with env and deployment files.

## Usage

Use this repo as a source of shared Cursor rules. Copy the rule files you want into a project's `.cursor/rules/` directory, or keep them here as a central reference.
